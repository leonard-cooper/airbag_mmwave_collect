uint32_t  num_chirps_per_frame,
uint32_t  num_samples_per_chirp 
)

Creates a frame structure for time domain real data acquisition.

This function initializes a data structure (of type ifx_Frame_R_t) that can hold a time domain data frame according to the dimensions provided as parameters. These parameters can be obtained for instance by parsing through a pre-recorded file. If a device is connected then the ifx_device_create_frame_from_device_handle should be used instead of this function, as that function reads the dimensions from configured the device handle.

Parameters
[in]num_antennasNumber of Rx antennas available in the device
[in]num_chirps_per_frameNumber of chirps configured in a frame
[in]num_samples_per_chirpNumber of samples per chirp
Returns
Pointer to allocated and initialized frame structure ifx_Frame_R_t or NULL if allocation failed.

◆ ifx_frame_destroy_r()

IFX_DLL_PUBLIC void ifx_frame_destroy_r ( ifx_Frame_R_t *  frame)

Frees the memory allocated for the real frame structure.

This function frees the memory associated with the provided frame structure and un-initializes it. This function must be called when the frame is not used any longer to avoid memory leaks. frame must not be used any more after this function was called.

This function frees the memory for the frame handle and the internal array of matrices.

Parameters
[in]frameThe real frame structure to be deallocated.

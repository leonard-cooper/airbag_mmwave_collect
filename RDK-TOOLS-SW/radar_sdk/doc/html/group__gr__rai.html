Calculates range angle image from complex (IQ) input raw data.

Parameters
[in]handleA handle to the Range Angle Image object
[in]inputAn array of complex 2D raw ADC data matrices for all Rx channels i.e. (Number of Antennas x NumChirps x Nsamples)
[out]outputAn arrays of real 2D matrices for range Doppler image beams i.e. (NumImages x Nsamples x NumberofBeams)

Detailed Description

API for Fast Fourier Transform (FFT)

Supports one-dimensional FFT for real and complex input signals.

IFX_DLL_PUBLIC ifx_FFT_Handle_t ifx_fft_create ( const ifx_FFT_Type_t  fft_type,
const uint32_t  fft_size 
)

Creates a FFT handle(object), based on the input parameters e.g.

FFT type (ifx_FFT_Type_t) and FFT size. This handle contains the complete twiddle table required for computing the FFT. During this handle construction, two cases have been taken into the consideration, regarding the input of the FFT as output is always complex.

  1. Real input FFT
  2. Complex input FFT In case 1, a special handle has been used to make FFT operation twice faster and memory efficient. The error code will be set if the fft_size parameter is outside the defined range of the ifx_FFT_Type_t enumeration.
    Note
    If the input vector is shorter than the FFT size, zero padding will automatically applied.
    If the input vector is longer than the FFT size, the FFT will only take FFT size initial samples of the input vector.
    Parameters
    [in]fft_typeFFT type is defined by ifx_FFT_Type_t. Type is defined based on FFT input nature, i.e. real or complex.
    [in]fft_sizeMust be power of 2 else returns error.
    Returns
    Handle to the newly created instance or NULL in case of failure.

ifx_Vector_C_t *  output

Performs shift on a FFT spectrum complex to bring DC bin in the center of spectrum, positive bins on right side and negative bins on left side.

Input and output memories should be of same type and size.

Parameters
[in]inputComplex array on which shift operation needs to be performed
[out]outputDC bin shifted to the center of the output array

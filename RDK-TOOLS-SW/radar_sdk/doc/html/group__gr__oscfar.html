Detailed Description

API for 2D ordered statistic constant false alarm rate (OS-CFAR) algorithm.

Input of this module is a 2D matrix of real values, i.e. range angle map. Output is the a 2D matrix of same size as Input with filtered contents.

An algorithm explanation is available at the Ordered Statistics Constant False Alarm Rate (OS-CFAR) SDK documentation.

Detailed Description

API for Mathematical operations.

Supports mathematical operations such as mean calculation, scalar removal from an array, element wise addition, subtraction and multiplication of arrays, scalar multiplication of an array, absolute value calculation, maximum value in an array.

Parameters
[in]inputReal value as input
[in]scaleFor voltage this should be 20 i.e. 20xlog10() and for power 10 i.e. 10xlog10() However, this is a generic math function, so scale can be any desired non zero float value
Returns

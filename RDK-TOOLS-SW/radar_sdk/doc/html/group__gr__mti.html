Creates a new MTI filter.

Instantiate an MTI filter ifx_MTI_Handle_t which contains information required to perform MTI (Moving Target Indication) filtering.

Parameters
[in]alpha_mti_filterFilter coefficient. Valid between 0.0 and 1.0
[in]spectrum_lengthLength of input / output vectors used within MTI filter
Returns
Handle to the newly created instance or NULL in case of failure.

◆ ifx_devconf_get_chirp_time()

IFX_DLL_PUBLIC ifx_Float_t ifx_devconf_get_chirp_time ( const ifx_Device_Config_t *  config)

Calculates the Chirp time in use from given device configuration.

The chirp time is the time where sampling is performed to get sampler per chirp. Initial chirp delays like PLL setup delay and ADC sampling delay are not used for chirp time calculations.

Parameters
[in]configDevice configuration structure
Returns
Chirp time (in seconds) used by device configuration
